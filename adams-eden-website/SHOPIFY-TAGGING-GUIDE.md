# Shopify Product Tagging Guide

This guide explains how to tag products in Shopify to ensure they appear in the correct categories on the Adams Eden website.

## How Categorization Works

Products are automatically categorized based on keywords found in:
- Product **Title**
- Product **Description**
- Product **Type**
- Product **Tags**

The system uses both **include keywords** (must match) and **exclude keywords** (must not match) to ensure accurate categorization.

## Best Practices

1. **Use Multiple Tags**: Add both general category tags (e.g., `flower`) and specific product tags (e.g., `marigold`) for better matching.

2. **Include "seed" or "seeds"**: For seed products, always include `seed` or `seeds` in tags or title to ensure they're categorized correctly.

3. **Be Specific**: Use the specific plant name in the product title or tags (e.g., "Tomato Seeds" should have `tomato` in title/tags).

4. **Avoid Conflicting Tags**: Don't mix plant categories (e.g., don't tag a flower as `vegetable`).

5. **Product Type Field**: Use the Product Type field in Shopify to add category information (e.g., "Flower Seeds", "Vegetable Seeds", "Herb Seeds").

6. **Test After Tagging**: After adding tags, check the website to ensure products appear in the correct categories.

## Edge Cases & Special Notes

### Products That Could Fit Multiple Categories

1. **Echinacea/Coneflower**: 
   - If sold as **herb/tea**: Tag with `herb`, `culinary`, `echinacea`
   - If sold as **ornamental flower**: Tag with `flower`, `perennial`, `coneflower`, `echinacea`

2. **Lavender**:
   - Can be both a **herb** (culinary) and a **flower** (ornamental)
   - Tag based on primary use: `herb` for culinary, `flower` + `perennial` for ornamental

3. **Sedum**:
   - Can be a **perennial flower** or a **succulent**
   - Tag based on variety: `flower` + `perennial` for garden sedums, `succulent` for stonecrop varieties

4. **Begonia**:
   - Can be an **annual flower** or a **houseplant**
   - Tag based on growing location: `annual` + `flower` for outdoor, `houseplant` for indoor

### Seed Products

All seed products should include `seed` or `seeds` in:
- The product title (e.g., "Tomato Seeds")
- Product tags
- Product type field (e.g., "Vegetable Seeds")

This ensures seeds are properly categorized even if the plant name isn't immediately obvious.

### Product Type Field Best Practices

Use descriptive product types in Shopify:

## Troubleshooting

**Product not showing in expected category?**
1. Check that the product title, description, or tags include the required keywords.
2. Verify that exclude keywords aren't present (e.g., a flower tagged with `vegetable` won't appear in Flowers).
3. Ensure the product is active and available for sale in Shopify.
4. Check that the product has been fetched by the website (products are cached; may need to wait for refresh).
5. Verify the product type field doesn't contain conflicting keywords.

**Product appearing in wrong category?**
1. Remove conflicting tags (e.g., remove `vegetable` from a flower product).
2. Add appropriate exclude keywords to the product description if needed.
3. Use more specific tags to improve matching accuracy.
4. Check if the product matches multiple categories - use more specific tags to narrow it down.

**Product appearing in multiple categories?**
- This is normal for some products (e.g., a product tagged with both `flower` and `perennial` will appear in both "Flowers" and "Flowers - Perennials").
- If you want a product to appear in only one category, use more specific tags and avoid general category tags.
